GENERAL NOTES
=============

A short summary of the file layout:

include/    Contains the minimum amount of C89-compatible header files necessary to use the library.
src/        Contains the C++ source code of all the projects.
premake/    Contains the premake executables, Lua script and batch/shell scripts to generate project files and compile them.
libs/       Contains the .dll and .lib files needed by the UDT_viewer_glfw project.
